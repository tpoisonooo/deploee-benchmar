# ax620a 测评

目录功能说明
```bash
.
├── docs               # 芯片 BSP 文档
├── hostcpu_test.md    # host CPU 峰值测试结果
├── model              # 模型转换和测试脚本，onnx 可从硬件模型库下载
├── model_test.md      # 模型测试结果报表
├── opr_test.md        # 算子测试结果报表
├── report.zip         # 模型和算子，执行期日志。文件中是 google drive 下载链接
├── scan_model.py      # 扫描模型日志、生成报表的脚本
└── scan_opr.py        # 扫描算子日志、生成报表的脚本
```
